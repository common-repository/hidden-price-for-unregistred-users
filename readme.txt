This plugin show price on products just for registred and loged users.

== Description ==
With this plugin you can hide prices to your woocommerce shop for products and it will be visible just for registred users.
Unregistred users will get notification message that you insert on settings page of plugin.

Available language :
English

To contribute in translating this plugin contact me : cvijetin.maletic@gmail.com


== Installation ==

1. Upload plugin to  "wp-content/plugins/" directory
2. Activate it in plugins section.

== Frequently Asked Questions ==

= How do I use the plugin? =
* Install and activate the plugin
* Go to Hidden Price For Unregistred on your Wordpress Dashboard -> in field "Notification Message" insert your text that will show on price place for unregistred users (exp. "Register on website to see the price").
